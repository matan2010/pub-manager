Pub Manager is a tablet-based Android application designed to help pubs and bars manage orders, products, families, and events in a simple and efficient way.

## Features
- Manage products and categories
- Create and manage orders
- Event-based order tracking
- Tablet-friendly UI built with Jetpack Compose
- Manual email sending for orders and reports

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM architecture
- Room Database
- Hilt (DI)
